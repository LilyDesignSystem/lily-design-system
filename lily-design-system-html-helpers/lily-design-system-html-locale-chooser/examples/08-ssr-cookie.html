Locale chooser — SSR pre-seed

Server resolved: locale=fr
Pre-rendered <html lang="fr" dir="ltr">

Why no flash?

  1. Server reads request.cookies.locale = fr.
  2. Layout emits <html lang="fr" dir="ltr">.
  3. Layout emits <locale-chooser value="fr">.
  4. Browser paints the page in French typography from the first byte.
  5. JS bundle loads, select upgrades, sees the inline value, skips localStorage and navigator resolution.
  6. Select re-applies lang="fr" (idempotent — no mutation).
  7. Future selection writes a fresh cookie.
