# BreadcrumbLink

One breadcrumb link in the trail.

See `components/breadcrumb-link/index.md` for canonical documentation.

## Parameters

- `Label`: string (optional) — accessible label set on `aria-label`
- `CssClass`: string — extra CSS classes appended to `breadcrumb-link`
- `ChildContent`: RenderFragment — component content
- `AdditionalAttributes`: catches unmatched HTML attributes

## Usage

```razor
<BreadcrumbLink>
    Content
</BreadcrumbLink>
```
