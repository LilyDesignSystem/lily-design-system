# CeskoRodneCisloInput

A an input for entering Czech Republic's Rodné číslo (RČ).

See `components/cesko-rodne-cislo-input/index.md` for canonical documentation.

## Parameters

- `Label`: string (required) — accessible label set on `aria-label`
- `CssClass`: string — extra CSS classes appended to `cesko-rodne-cislo-input`

- `AdditionalAttributes`: catches unmatched HTML attributes

## Usage

```razor
<CeskoRodneCisloInput Label="Czech Personal Number" />
```
