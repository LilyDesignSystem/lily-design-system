# NorgeFodselsnummerInput

A an input for entering Norway's Fødselsnummer.

See `components/norge-fodselsnummer-input/index.md` for canonical documentation.

## Parameters

- `Label`: string (required) — accessible label set on `aria-label`
- `CssClass`: string — extra CSS classes appended to `norge-fodselsnummer-input`

- `AdditionalAttributes`: catches unmatched HTML attributes

## Usage

```razor
<NorgeFodselsnummerInput Label="Norwegian Birth Number" />
```
