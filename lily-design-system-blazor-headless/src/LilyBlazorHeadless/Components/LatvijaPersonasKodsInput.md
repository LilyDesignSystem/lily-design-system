# LatvijaPersonasKodsInput

A an input for entering Latvia's Personas kods.

See `components/latvija-personas-kods-input/index.md` for canonical documentation.

## Parameters

- `Label`: string (required) — accessible label set on `aria-label`
- `CssClass`: string — extra CSS classes appended to `latvija-personas-kods-input`

- `AdditionalAttributes`: catches unmatched HTML attributes

## Usage

```razor
<LatvijaPersonasKodsInput Label="Personal Code" />
```
