# SlovenijaEmsoInput

A an input for entering Slovenia's Enotna Matična Številka Občana (EMŠO).

See `components/slovenija-emso-input/index.md` for canonical documentation.

## Parameters

- `Label`: string (required) — accessible label set on `aria-label`
- `CssClass`: string — extra CSS classes appended to `slovenija-emso-input`

- `AdditionalAttributes`: catches unmatched HTML attributes

## Usage

```razor
<SlovenijaEmsoInput Label="Unique Master Citizen Number" />
```
