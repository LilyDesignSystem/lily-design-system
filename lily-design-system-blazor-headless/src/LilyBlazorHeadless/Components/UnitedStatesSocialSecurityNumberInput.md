# UnitedStatesSocialSecurityNumberInput

An input for entering a US Social Security number.

See `components/united-states-social-security-number-input/index.md` for canonical documentation.

## Parameters

- `Label`: string (required) — accessible label set on `aria-label`
- `CssClass`: string — extra CSS classes appended to `united-states-social-security-number-input`
- `ChildContent`: RenderFragment — component content
- `AdditionalAttributes`: catches unmatched HTML attributes

## Usage

```razor
<UnitedStatesSocialSecurityNumberInput Label="...">
    Content
</UnitedStatesSocialSecurityNumberInput>
```
