# RomaniaPasaportInput

A an input for entering Romania's Paşaport.

See `components/romania-pasaport-input/index.md` for canonical documentation.

## Parameters

- `Label`: string (required) — accessible label set on `aria-label`
- `CssClass`: string — extra CSS classes appended to `romania-pasaport-input`

- `AdditionalAttributes`: catches unmatched HTML attributes

## Usage

```razor
<RomaniaPasaportInput Label="Romanian Passport" />
```
