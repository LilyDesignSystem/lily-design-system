# LietuvaPasasInput

A an input for entering Lithuania's Pasas.

See `components/lietuva-pasas-input/index.md` for canonical documentation.

## Parameters

- `Label`: string (required) — accessible label set on `aria-label`
- `CssClass`: string — extra CSS classes appended to `lietuva-pasas-input`

- `AdditionalAttributes`: catches unmatched HTML attributes

## Usage

```razor
<LietuvaPasasInput Label="Lithuanian Passport" />
```
