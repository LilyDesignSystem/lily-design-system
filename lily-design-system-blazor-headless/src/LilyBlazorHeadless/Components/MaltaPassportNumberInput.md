# MaltaPassportNumberInput

A an input for entering Malta's Malta Passport Number.

See `components/malta-passport-number-input/index.md` for canonical documentation.

## Parameters

- `Label`: string (required) — accessible label set on `aria-label`
- `CssClass`: string — extra CSS classes appended to `malta-passport-number-input`

- `AdditionalAttributes`: catches unmatched HTML attributes

## Usage

```razor
<MaltaPassportNumberInput Label="Malta Passport Number" />
```
