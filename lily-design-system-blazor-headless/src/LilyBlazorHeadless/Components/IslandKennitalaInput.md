# IslandKennitalaInput

A an input for entering Iceland's Kennitala.

See `components/island-kennitala-input/index.md` for canonical documentation.

## Parameters

- `Label`: string (required) — accessible label set on `aria-label`
- `CssClass`: string — extra CSS classes appended to `island-kennitala-input`

- `AdditionalAttributes`: catches unmatched HTML attributes

## Usage

```razor
<IslandKennitalaInput Label="Personal Identity Code" />
```
