# GanttTableTfoot

A Gantt chart table interactive grid tfoot for planning schedule visualization <tfoot>.

See `components/gantt-table-tfoot/index.md` for canonical documentation.

## Parameters

- `Label`: string (optional) — accessible label set on `aria-label`
- `CssClass`: string — extra CSS classes appended to `gantt-table-tfoot`
- `ChildContent`: RenderFragment — component content
- `AdditionalAttributes`: catches unmatched HTML attributes

## Usage

```razor
<GanttTableTfoot>
    Content
</GanttTableTfoot>
```
