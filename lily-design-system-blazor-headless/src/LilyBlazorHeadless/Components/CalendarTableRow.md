# CalendarTableRow

A calendar table interactive grid row for managing dates, days, etc. <tr>.

See `components/calendar-table-row/index.md` for canonical documentation.

## Parameters

- `Label`: string (optional) — accessible label set on `aria-label`
- `CssClass`: string — extra CSS classes appended to `calendar-table-row`
- `ChildContent`: RenderFragment — component content
- `AdditionalAttributes`: catches unmatched HTML attributes

## Usage

```razor
<CalendarTableRow>
    Content
</CalendarTableRow>
```
