# NederlandIdentiteitskaartNummerInput

A an input for entering Netherlands's Identiteitskaart Nummer.

See `components/nederland-identiteitskaart-nummer-input/index.md` for canonical documentation.

## Parameters

- `Label`: string (required) — accessible label set on `aria-label`
- `CssClass`: string — extra CSS classes appended to `nederland-identiteitskaart-nummer-input`

- `AdditionalAttributes`: catches unmatched HTML attributes

## Usage

```razor
<NederlandIdentiteitskaartNummerInput Label="Dutch National Identity Card Number" />
```
