# SlovenskoPasInput

A an input for entering Slovakia's Pas.

See `components/slovensko-pas-input/index.md` for canonical documentation.

## Parameters

- `Label`: string (required) — accessible label set on `aria-label`
- `CssClass`: string — extra CSS classes appended to `slovensko-pas-input`

- `AdditionalAttributes`: catches unmatched HTML attributes

## Usage

```razor
<SlovenskoPasInput Label="Slovak Passport" />
```
