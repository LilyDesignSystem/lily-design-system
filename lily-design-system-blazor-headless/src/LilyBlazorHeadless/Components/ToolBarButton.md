# ToolBarButton

One action button in a tool bar.

See `components/tool-bar-button/index.md` for canonical documentation.

## Parameters

- `Label`: string (optional) — accessible label set on `aria-label`
- `CssClass`: string — extra CSS classes appended to `tool-bar-button`
- `ChildContent`: RenderFragment — component content
- `AdditionalAttributes`: catches unmatched HTML attributes

## Usage

```razor
<ToolBarButton>
    Content
</ToolBarButton>
```
