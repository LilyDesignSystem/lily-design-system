# DanmarkPersonnummerInput

A an input for entering Denmark's Personnummer (CPR-nummer) (CPR).

See `components/danmark-personnummer-input/index.md` for canonical documentation.

## Parameters

- `Label`: string (required) — accessible label set on `aria-label`
- `CssClass`: string — extra CSS classes appended to `danmark-personnummer-input`

- `AdditionalAttributes`: catches unmatched HTML attributes

## Usage

```razor
<DanmarkPersonnummerInput Label="Personal Identity Code" />
```
