# MenuBarButton

One item in a menu bar.

See `components/menu-bar-button/index.md` for canonical documentation.

## Parameters

- `Label`: string (optional) — accessible label set on `aria-label`
- `CssClass`: string — extra CSS classes appended to `menu-bar-button`
- `ChildContent`: RenderFragment — component content
- `AdditionalAttributes`: catches unmatched HTML attributes

## Usage

```razor
<MenuBarButton>
    Content
</MenuBarButton>
```
