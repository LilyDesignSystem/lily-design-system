# BelgiqueNumeroDeRegistreNationalInput

A an input for entering Belgium's Numéro de Registre National / Rijksregisternummer (NRN).

See `components/belgique-numero-de-registre-national-input/index.md` for canonical documentation.

## Parameters

- `Label`: string (required) — accessible label set on `aria-label`
- `CssClass`: string — extra CSS classes appended to `belgique-numero-de-registre-national-input`

- `AdditionalAttributes`: catches unmatched HTML attributes

## Usage

```razor
<BelgiqueNumeroDeRegistreNationalInput Label="National Register Number" />
```
