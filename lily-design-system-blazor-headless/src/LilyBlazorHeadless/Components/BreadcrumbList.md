# BreadcrumbList

An ordered list of breadcrumb list items.

See `components/breadcrumb-list/index.md` for canonical documentation.

## Parameters

- `Label`: string (optional) — accessible label set on `aria-label`
- `CssClass`: string — extra CSS classes appended to `breadcrumb-list`
- `ChildContent`: RenderFragment — component content
- `AdditionalAttributes`: catches unmatched HTML attributes

## Usage

```razor
<BreadcrumbList>
    Content
</BreadcrumbList>
```
