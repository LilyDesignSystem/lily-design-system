# GanttTableTH

A Gantt chart table interactive grid header cell for planning schedule visualization <th>.

See `components/gantt-table-th/index.md` for canonical documentation.

## Parameters

- `Label`: string (optional) — accessible label set on `aria-label`
- `CssClass`: string — extra CSS classes appended to `gantt-table-th`
- `ChildContent`: RenderFragment — component content
- `AdditionalAttributes`: catches unmatched HTML attributes

## Usage

```razor
<GanttTableTH>
    Content
</GanttTableTH>
```
