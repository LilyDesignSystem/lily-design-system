# RedAmberGreenPicker

A picker for selecting a red/amber/green status.

See `components/red-amber-green-picker/index.md` for canonical documentation.

## Parameters

- `Label`: string (required) — accessible label set on `aria-label`
- `CssClass`: string — extra CSS classes appended to `red-amber-green-picker`
- `ChildContent`: RenderFragment — component content
- `AdditionalAttributes`: catches unmatched HTML attributes

## Usage

```razor
<RedAmberGreenPicker Label="...">
    Content
</RedAmberGreenPicker>
```
