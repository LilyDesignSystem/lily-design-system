# UnitedKingdomNationalInsuranceNumberInput

A an input for entering United Kingdom's National Insurance Number (NINO).

See `components/united-kingdom-national-insurance-number-input/index.md` for canonical documentation.

## Parameters

- `Label`: string (required) — accessible label set on `aria-label`
- `CssClass`: string — extra CSS classes appended to `united-kingdom-national-insurance-number-input`

- `AdditionalAttributes`: catches unmatched HTML attributes

## Usage

```razor
<UnitedKingdomNationalInsuranceNumberInput Label="National Insurance Number" />
```
