# UnitedKingdomNationalHealthServiceNumberView

A read-only display of a UK NHS number.

See `components/united-kingdom-national-health-service-number-view/index.md` for canonical documentation.

## Parameters

- `Label`: string (required) — accessible label set on `aria-label`
- `CssClass`: string — extra CSS classes appended to `united-kingdom-national-health-service-number-view`
- `ChildContent`: RenderFragment — component content
- `AdditionalAttributes`: catches unmatched HTML attributes

## Usage

```razor
<UnitedKingdomNationalHealthServiceNumberView Label="...">
    Content
</UnitedKingdomNationalHealthServiceNumberView>
```
