# UnitedKingdomNationalHealthServiceNumberInput

An input for entering a UK NHS number.

See `components/united-kingdom-national-health-service-number-input/index.md` for canonical documentation.

## Parameters

- `Label`: string (required) — accessible label set on `aria-label`
- `CssClass`: string — extra CSS classes appended to `united-kingdom-national-health-service-number-input`
- `ChildContent`: RenderFragment — component content
- `AdditionalAttributes`: catches unmatched HTML attributes

## Usage

```razor
<UnitedKingdomNationalHealthServiceNumberInput Label="...">
    Content
</UnitedKingdomNationalHealthServiceNumberInput>
```
