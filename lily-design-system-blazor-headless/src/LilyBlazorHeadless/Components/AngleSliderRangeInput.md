# AngleSliderRangeInput

A range input for selecting an angle in degrees.

See `components/angle-slider-range-input/index.md` for canonical documentation.

## Parameters

- `Label`: string (required) — accessible label set on `aria-label`
- `CssClass`: string — extra CSS classes appended to `angle-slider-range-input`
- `ChildContent`: RenderFragment — component content
- `AdditionalAttributes`: catches unmatched HTML attributes

## Usage

```razor
<AngleSliderRangeInput Label="...">
    Content
</AngleSliderRangeInput>
```
