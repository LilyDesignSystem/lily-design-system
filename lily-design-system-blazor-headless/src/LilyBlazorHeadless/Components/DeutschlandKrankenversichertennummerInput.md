# DeutschlandKrankenversichertennummerInput

A an input for entering Germany's Krankenversichertennummer (KVNR).

See `components/deutschland-krankenversichertennummer-input/index.md` for canonical documentation.

## Parameters

- `Label`: string (required) — accessible label set on `aria-label`
- `CssClass`: string — extra CSS classes appended to `deutschland-krankenversichertennummer-input`

- `AdditionalAttributes`: catches unmatched HTML attributes

## Usage

```razor
<DeutschlandKrankenversichertennummerInput Label="Health Insurance Number" />
```
