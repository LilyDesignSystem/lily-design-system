# CeskoCestovniPasInput

A an input for entering Czech Republic's Cestovní pas.

See `components/cesko-cestovni-pas-input/index.md` for canonical documentation.

## Parameters

- `Label`: string (required) — accessible label set on `aria-label`
- `CssClass`: string — extra CSS classes appended to `cesko-cestovni-pas-input`

- `AdditionalAttributes`: catches unmatched HTML attributes

## Usage

```razor
<CeskoCestovniPasInput Label="Czech Passport" />
```
