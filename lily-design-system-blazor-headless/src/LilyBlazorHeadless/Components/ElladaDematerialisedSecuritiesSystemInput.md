# ElladaDematerialisedSecuritiesSystemInput

A an input for entering Greece's Dematerialised Securities System (DSS).

See `components/ellada-dematerialised-securities-system-input/index.md` for canonical documentation.

## Parameters

- `Label`: string (required) — accessible label set on `aria-label`
- `CssClass`: string — extra CSS classes appended to `ellada-dematerialised-securities-system-input`

- `AdditionalAttributes`: catches unmatched HTML attributes

## Usage

```razor
<ElladaDematerialisedSecuritiesSystemInput Label="Dematerialised Securities System" />
```
