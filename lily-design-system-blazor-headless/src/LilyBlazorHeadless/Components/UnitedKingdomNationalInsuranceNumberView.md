# UnitedKingdomNationalInsuranceNumberView

A a read-only display of United Kingdom's National Insurance Number (NINO).

See `components/united-kingdom-national-insurance-number-view/index.md` for canonical documentation.

## Parameters

- `Label`: string (required) — accessible label set on `aria-label`
- `CssClass`: string — extra CSS classes appended to `united-kingdom-national-insurance-number-view`
- `ChildContent`: RenderFragment — component content
- `AdditionalAttributes`: catches unmatched HTML attributes

## Usage

```razor
<UnitedKingdomNationalInsuranceNumberView Label="National Insurance Number">...</UnitedKingdomNationalInsuranceNumberView>
```
