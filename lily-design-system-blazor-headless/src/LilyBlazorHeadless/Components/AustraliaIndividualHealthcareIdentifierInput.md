# AustraliaIndividualHealthcareIdentifierInput

A an input for entering Australia's Individual Healthcare Identifier (IHI).

See `components/australia-individual-healthcare-identifier-input/index.md` for canonical documentation.

## Parameters

- `Label`: string (required) — accessible label set on `aria-label`
- `CssClass`: string — extra CSS classes appended to `australia-individual-healthcare-identifier-input`

- `AdditionalAttributes`: catches unmatched HTML attributes

## Usage

```razor
<AustraliaIndividualHealthcareIdentifierInput Label="Individual Healthcare Identifier" />
```
