# TaskBarButton

One item in a task bar.

See `components/task-bar-button/index.md` for canonical documentation.

## Parameters

- `Label`: string (optional) — accessible label set on `aria-label`
- `CssClass`: string — extra CSS classes appended to `task-bar-button`
- `ChildContent`: RenderFragment — component content
- `AdditionalAttributes`: catches unmatched HTML attributes

## Usage

```razor
<TaskBarButton>
    Content
</TaskBarButton>
```
