# NederlandPaspoortNummerInput

A an input for entering Netherlands's Paspoort Nummer.

See `components/nederland-paspoort-nummer-input/index.md` for canonical documentation.

## Parameters

- `Label`: string (required) — accessible label set on `aria-label`
- `CssClass`: string — extra CSS classes appended to `nederland-paspoort-nummer-input`

- `AdditionalAttributes`: catches unmatched HTML attributes

## Usage

```razor
<NederlandPaspoortNummerInput Label="Dutch Passport Number" />
```
