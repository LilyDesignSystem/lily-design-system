# FranceNumeroDIdentificationAuRepertoireInput

An input for entering a France numéro d'identification au répertoire (NIR) unique national healthcare identifier.

See `components/france-numero-d-identification-au-repertoire-input/index.md` for canonical documentation.

## Parameters

- `Label`: string (required) — accessible label set on `aria-label`
- `CssClass`: string — extra CSS classes appended to `france-numero-d-identification-au-repertoire-input`
- `ChildContent`: RenderFragment — component content
- `AdditionalAttributes`: catches unmatched HTML attributes

## Usage

```razor
<FranceNumeroDIdentificationAuRepertoireInput Label="...">
    Content
</FranceNumeroDIdentificationAuRepertoireInput>
```
