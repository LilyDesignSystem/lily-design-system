# LiechtensteinPassportNumberInput

A an input for entering Liechtenstein's Liechtenstein Passport Number.

See `components/liechtenstein-passport-number-input/index.md` for canonical documentation.

## Parameters

- `Label`: string (required) — accessible label set on `aria-label`
- `CssClass`: string — extra CSS classes appended to `liechtenstein-passport-number-input`

- `AdditionalAttributes`: catches unmatched HTML attributes

## Usage

```razor
<LiechtensteinPassportNumberInput Label="Liechtenstein Passport Number" />
```
