# AlbaCommunityHealthIndexInput

A an input for entering Scotland's Community Health Index (CHI).

See `components/alba-community-health-index-input/index.md` for canonical documentation.

## Parameters

- `Label`: string (required) — accessible label set on `aria-label`
- `CssClass`: string — extra CSS classes appended to `alba-community-health-index-input`

- `AdditionalAttributes`: catches unmatched HTML attributes

## Usage

```razor
<AlbaCommunityHealthIndexInput Label="Community Health Index" />
```
