# RedAmberGreenPickerButton

A picker button for selecting a red/amber/green status.

See `components/red-amber-green-picker-button/index.md` for canonical documentation.

## Parameters

- `Label`: string (required) — accessible label set on `aria-label`
- `CssClass`: string — extra CSS classes appended to `red-amber-green-picker-button`
- `ChildContent`: RenderFragment — component content
- `AdditionalAttributes`: catches unmatched HTML attributes

## Usage

```razor
<RedAmberGreenPickerButton Label="...">
    Content
</RedAmberGreenPickerButton>
```
