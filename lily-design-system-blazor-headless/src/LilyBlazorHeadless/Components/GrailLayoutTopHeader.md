# GrailLayoutTopHeader

Grail layout top header full width with <header> tag.

See `components/grail-layout-top-header/index.md` for canonical documentation.

## Parameters

- `Label`: string (optional) — accessible label set on `aria-label`
- `CssClass`: string — extra CSS classes appended to `grail-layout-top-header`
- `ChildContent`: RenderFragment — component content
- `AdditionalAttributes`: catches unmatched HTML attributes

## Usage

```razor
<GrailLayoutTopHeader>
    Content
</GrailLayoutTopHeader>
```
