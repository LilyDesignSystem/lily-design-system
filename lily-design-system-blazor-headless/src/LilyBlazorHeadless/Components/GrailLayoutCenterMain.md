# GrailLayoutCenterMain

Grail layout center main content with <main> tag.

See `components/grail-layout-center-main/index.md` for canonical documentation.

## Parameters

- `Label`: string (optional) — accessible label set on `aria-label`
- `CssClass`: string — extra CSS classes appended to `grail-layout-center-main`
- `ChildContent`: RenderFragment — component content
- `AdditionalAttributes`: catches unmatched HTML attributes

## Usage

```razor
<GrailLayoutCenterMain>
    Content
</GrailLayoutCenterMain>
```
