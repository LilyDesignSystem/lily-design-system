# PortugalNumeroDeIdentificacaoFiscalInput

A an input for entering Portugal's Número de Identificação Fiscal (NIF).

See `components/portugal-numero-de-identificacao-fiscal-input/index.md` for canonical documentation.

## Parameters

- `Label`: string (required) — accessible label set on `aria-label`
- `CssClass`: string — extra CSS classes appended to `portugal-numero-de-identificacao-fiscal-input`

- `AdditionalAttributes`: catches unmatched HTML attributes

## Usage

```razor
<PortugalNumeroDeIdentificacaoFiscalInput Label="Tax Identification Number" />
```
