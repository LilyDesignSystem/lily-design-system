# SegmentGroupItem

One selectable segment in a segment group.

See `components/segment-group-item/index.md` for canonical documentation.

## Parameters

- `Label`: string (optional) — accessible label set on `aria-label`
- `CssClass`: string — extra CSS classes appended to `segment-group-item`
- `ChildContent`: RenderFragment — component content
- `AdditionalAttributes`: catches unmatched HTML attributes

## Usage

```razor
<SegmentGroupItem>
    Content
</SegmentGroupItem>
```
