# NederlandBurgerserviceNummerInput

A an input for entering Netherlands's Burgerservicenummer (BSN).

See `components/nederland-burgerservice-nummer-input/index.md` for canonical documentation.

## Parameters

- `Label`: string (required) — accessible label set on `aria-label`
- `CssClass`: string — extra CSS classes appended to `nederland-burgerservice-nummer-input`

- `AdditionalAttributes`: catches unmatched HTML attributes

## Usage

```razor
<NederlandBurgerserviceNummerInput Label="Citizen Service Number" />
```
