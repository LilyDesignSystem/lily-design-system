# EnglandNationalHealthServiceNumberInput

A an input for entering England's National Health Service Number (NHS).

See `components/england-national-health-service-number-input/index.md` for canonical documentation.

## Parameters

- `Label`: string (required) — accessible label set on `aria-label`
- `CssClass`: string — extra CSS classes appended to `england-national-health-service-number-input`

- `AdditionalAttributes`: catches unmatched HTML attributes

## Usage

```razor
<EnglandNationalHealthServiceNumberInput Label="National Health Service Number" />
```
