# MaltaNationalIdentificationNumberInput

A an input for entering Malta's Malta National Identification Number.

See `components/malta-national-identification-number-input/index.md` for canonical documentation.

## Parameters

- `Label`: string (required) — accessible label set on `aria-label`
- `CssClass`: string — extra CSS classes appended to `malta-national-identification-number-input`

- `AdditionalAttributes`: catches unmatched HTML attributes

## Usage

```razor
<MaltaNationalIdentificationNumberInput Label="Malta National Identification Number" />
```
