# ContainerWithFluidWidth

A full-width content wrapper with horizontal padding.

See `components/container-with-fluid-width/index.md` for canonical documentation.

## Parameters

- `Label`: string (optional) — accessible label set on `aria-label`
- `CssClass`: string — extra CSS classes appended to `container-with-fluid-width`
- `ChildContent`: RenderFragment — component content
- `AdditionalAttributes`: catches unmatched HTML attributes

## Usage

```razor
<ContainerWithFluidWidth>
    Content
</ContainerWithFluidWidth>
```
