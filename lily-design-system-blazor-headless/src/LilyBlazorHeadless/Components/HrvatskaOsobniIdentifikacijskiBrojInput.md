# HrvatskaOsobniIdentifikacijskiBrojInput

A an input for entering Croatia's Osobni identifikacijski broj (OIB).

See `components/hrvatska-osobni-identifikacijski-broj-input/index.md` for canonical documentation.

## Parameters

- `Label`: string (required) — accessible label set on `aria-label`
- `CssClass`: string — extra CSS classes appended to `hrvatska-osobni-identifikacijski-broj-input`

- `AdditionalAttributes`: catches unmatched HTML attributes

## Usage

```razor
<HrvatskaOsobniIdentifikacijskiBrojInput Label="Personal Identification Number" />
```
