# TabBarButton

One tab button in a tab group.

See `components/tab-bar-button/index.md` for canonical documentation.

## Parameters

- `Label`: string (optional) — accessible label set on `aria-label`
- `CssClass`: string — extra CSS classes appended to `tab-bar-button`
- `ChildContent`: RenderFragment — component content
- `AdditionalAttributes`: catches unmatched HTML attributes

## Usage

```razor
<TabBarButton>
    Content
</TabBarButton>
```
