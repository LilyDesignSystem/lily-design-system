# GanttTableThead

A Gantt chart table interactive grid thead for planning schedule visualization <thead>.

See `components/gantt-table-thead/index.md` for canonical documentation.

## Parameters

- `Label`: string (optional) — accessible label set on `aria-label`
- `CssClass`: string — extra CSS classes appended to `gantt-table-thead`
- `ChildContent`: RenderFragment — component content
- `AdditionalAttributes`: catches unmatched HTML attributes

## Usage

```razor
<GanttTableThead>
    Content
</GanttTableThead>
```
