# EestiIsikukoodInput

A an input for entering Estonia's Isikukood (IK).

See `components/eesti-isikukood-input/index.md` for canonical documentation.

## Parameters

- `Label`: string (required) — accessible label set on `aria-label`
- `CssClass`: string — extra CSS classes appended to `eesti-isikukood-input`

- `AdditionalAttributes`: catches unmatched HTML attributes

## Usage

```razor
<EestiIsikukoodInput Label="Personal Identification Code" />
```
