# SverigePersonnummerInput

A an input for entering Sweden's Personnummer.

See `components/sverige-personnummer-input/index.md` for canonical documentation.

## Parameters

- `Label`: string (required) — accessible label set on `aria-label`
- `CssClass`: string — extra CSS classes appended to `sverige-personnummer-input`

- `AdditionalAttributes`: catches unmatched HTML attributes

## Usage

```razor
<SverigePersonnummerInput Label="Personal Identity Number" />
```
