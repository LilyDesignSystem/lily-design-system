# SlovenskoRodneCisloInput

A an input for entering Slovakia's Rodné číslo (RČ).

See `components/slovensko-rodne-cislo-input/index.md` for canonical documentation.

## Parameters

- `Label`: string (required) — accessible label set on `aria-label`
- `CssClass`: string — extra CSS classes appended to `slovensko-rodne-cislo-input`

- `AdditionalAttributes`: catches unmatched HTML attributes

## Usage

```razor
<SlovenskoRodneCisloInput Label="Slovak Personal Number" />
```
