# CalendarTableFoot

A calendar table interactive grid tfoot for managing dates, days, etc. <tfoot>.

See `components/calendar-table-foot/index.md` for canonical documentation.

## Parameters

- `Label`: string (optional) — accessible label set on `aria-label`
- `CssClass`: string — extra CSS classes appended to `calendar-table-foot`
- `ChildContent`: RenderFragment — component content
- `AdditionalAttributes`: catches unmatched HTML attributes

## Usage

```razor
<CalendarTableFoot>
    Content
</CalendarTableFoot>
```
