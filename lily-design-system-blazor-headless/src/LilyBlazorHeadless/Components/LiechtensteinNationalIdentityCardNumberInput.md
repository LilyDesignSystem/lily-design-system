# LiechtensteinNationalIdentityCardNumberInput

A an input for entering Liechtenstein's Liechtenstein National Identity Card Number.

See `components/liechtenstein-national-identity-card-number-input/index.md` for canonical documentation.

## Parameters

- `Label`: string (required) — accessible label set on `aria-label`
- `CssClass`: string — extra CSS classes appended to `liechtenstein-national-identity-card-number-input`

- `AdditionalAttributes`: catches unmatched HTML attributes

## Usage

```razor
<LiechtensteinNationalIdentityCardNumberInput Label="Liechtenstein National Identity Card Number" />
```
