# PolskaNumerIdentyfikacjiPodatkowejInput

A an input for entering Poland's Numer Identyfikacji Podatkowej (NIP).

See `components/polska-numer-identyfikacji-podatkowej-input/index.md` for canonical documentation.

## Parameters

- `Label`: string (required) — accessible label set on `aria-label`
- `CssClass`: string — extra CSS classes appended to `polska-numer-identyfikacji-podatkowej-input`

- `AdditionalAttributes`: catches unmatched HTML attributes

## Usage

```razor
<PolskaNumerIdentyfikacjiPodatkowejInput Label="Tax Identification Number" />
```
