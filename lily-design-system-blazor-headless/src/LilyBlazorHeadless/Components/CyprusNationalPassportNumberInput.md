# CyprusNationalPassportNumberInput

A an input for entering Cyprus's National Passport Number.

See `components/cyprus-national-passport-number-input/index.md` for canonical documentation.

## Parameters

- `Label`: string (required) — accessible label set on `aria-label`
- `CssClass`: string — extra CSS classes appended to `cyprus-national-passport-number-input`

- `AdditionalAttributes`: catches unmatched HTML attributes

## Usage

```razor
<CyprusNationalPassportNumberInput Label="National Passport Number" />
```
