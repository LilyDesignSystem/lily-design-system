# ThemeSelectOption

One option in a theme select dropdown.

See `components/theme-select-option/index.md` for canonical documentation.

## Parameters

- `Label`: string (optional) — accessible label set on `aria-label`
- `CssClass`: string — extra CSS classes appended to `theme-select-option`
- `ChildContent`: RenderFragment — component content
- `AdditionalAttributes`: catches unmatched HTML attributes

## Usage

```razor
<ThemeSelectOption>
    Content
</ThemeSelectOption>
```
