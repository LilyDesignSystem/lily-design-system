# GrailLayoutLeftAside

Grail layout left aside sidebar with <aside> tag.

See `components/grail-layout-left-aside/index.md` for canonical documentation.

## Parameters

- `Label`: string (optional) — accessible label set on `aria-label`
- `CssClass`: string — extra CSS classes appended to `grail-layout-left-aside`
- `ChildContent`: RenderFragment — component content
- `AdditionalAttributes`: catches unmatched HTML attributes

## Usage

```razor
<GrailLayoutLeftAside>
    Content
</GrailLayoutLeftAside>
```
