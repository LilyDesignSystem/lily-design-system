# CymruRhifYGwasanaethIechydGwladolInput

A an input for entering Wales's Rhif y Gwasanaeth Iechyd Gwladol (Rhif GIG).

See `components/cymru-rhif-y-gwasanaeth-iechyd-gwladol-input/index.md` for canonical documentation.

## Parameters

- `Label`: string (required) — accessible label set on `aria-label`
- `CssClass`: string — extra CSS classes appended to `cymru-rhif-y-gwasanaeth-iechyd-gwladol-input`

- `AdditionalAttributes`: catches unmatched HTML attributes

## Usage

```razor
<CymruRhifYGwasanaethIechydGwladolInput Label="National Health Service Number" />
```
